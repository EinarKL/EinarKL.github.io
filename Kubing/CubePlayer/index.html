<cube-player> optional attributes


Add this line to your HTML document to use <cube-player>:

<script src="https://einarkl.io/Kubing/CubePlayer.js" type="module"></script>


As of today's date only 1 occurence of <cube-player> is supported.

attribute value options
scramble scramble
solution solution
time time in ms
cubestyle solid | stickerless | hollow
logo url to image file
colors 6 hex colors separated by , in order U L F R B D
plastic hex color
